Summary

With such a concentrated market among cloud storage applications it’s difficult for users to select one application that meets all their personal and professional needs.These existing platforms are coming up short when it comes to updated designs, specific collaboration features, and content sharing methodologies.

Problem

Many of the cloud storage applications today often specialize in certain features and lack in others. The result is frustrated users having to utilize multiple applications to achieve different tasks. RainDrops strives to connect it’s users to their everyday functions all in one drop.

Solution

Through extensive research, user testing, and a simplistic design approach, I developed Raindrops. A cloud storage application that allows users to save, share, and collaborate all from one central location.

Challenges
